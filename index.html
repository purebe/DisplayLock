Displaylock

DisplayLock is a simple program that when run attempts to turn off your display and lock your computer.

Download this project as a .zip file Download this project as a tar.gz file

The Goal

This simple program was designed because my computer is in the same room where I sleep. Occasionally, I have need to keep my computer on all night and it's annoying waiting for the display to automatically turn off when I head for bed. With this program, I simply run it and can rest assured knowing my computer is both locked, and not ruining my sleep by illuminating the room.
